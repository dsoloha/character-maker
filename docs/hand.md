# `.hand`

`class Hand`

## Description

Properties pertaining to the hand on the arm.

## Declaration

```js
import { Hand } from 'character-maker'

const hand = new Hand()

// or optionally with parameters:

const hand = new Hand({
  fingers: 4,
})
```

## Properties

| Property  | Type     | Description                    | Optional |
| :-------- | :------- | :----------------------------- | :------- |
| `fingers` | `number` | How many fingers the hand has. | Yes      |
