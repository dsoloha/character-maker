# `.stomach`

## Description

```js
import { Stomach } from 'character-maker'

const stomach = new Stomach()

const stomach = new Stomach({

})
```

## Properties

| Property | Type     | Description                           | Optional |
| :------- | :------- | :------------------------------------ | :------- |
| `shape`  | `string` | The shape of the character's stomach. | Yes      |
